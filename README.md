# Bridge_Docs_Obsidian
local docs in a git repo for collaborations


**This setup should and will automatically synchronised to everyone you just need to import the repo and simply open the folder as a vault inside obsidian**

>[!important]
>this is a common setup for all so don't do any changes that could cause RAM usage for all users keep it minimal and don't install any unnecessary plugins
> - use " / "  for available actions also can use **CTRL + P** for command  palettes 
> - when online the repo will **automatically pulled at startup** and the recent changes will be synced
> -  it will also commit and push **2 minutes** after changing the documents  
## ==   **_Do Not  delete any file accidentally !!!_**  ==


%% step one: 
- install obsidian
- clone this repo to a specific folder
- import this folder as vault in obsidian
- settings >  Files and Links > turn on automatically update internal links
- enable community plugins in settings 
- install GIT plugin from the community plugins page - obsidian://show-plugin?id=obsidian-git
- enable the plugins
- first pull and refresh for any changes in the repo by clicking on pull in the Git plugin panel
- if made changes do commit and sync in the GIT plugin panel
- one changes at a time to avoid conflicts
- **don't delete any file** accidentally
