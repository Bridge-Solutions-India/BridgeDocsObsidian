**new react application in current folder**
```
npm create vite@latest .
```

**Dependencies**
1. tailwindcss
```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
2. shadcn UI library
```
npm install shadcn-ui
npx shadcn-ui init
```

**run react development app**
```
npm run
```
